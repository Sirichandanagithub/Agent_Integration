# 🤖 Enterprise AI Assistant

An **Agentic AI-powered ServiceNow chatbot** that intelligently answers IT incident-related questions using **LangGraph, LangChain, Gemini, and ServiceNow APIs**.

This project enables users to interact with ServiceNow incident data in natural language, while an intelligent routing system decides which specialist agent should handle the request.

---
